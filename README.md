# Gator Advisor

### Running the Project

To run the backend, navigate to /src/backend/, run:

### `node server.js`

To run the frontend, navigate to the project directory, run:

### `npm start`

Open [http://localhost:3000](http://localhost:3000) to view it in your browser.
